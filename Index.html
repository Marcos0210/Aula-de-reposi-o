Sobre o HTML

HTML

HTML (HyperText Markup Language) é a linguagem padrão para criar páginas da web. Ela é usada para estruturar o conteúdo na web, permitindo que os desenvolvedores definam elementos como cabeçalhos, parágrafos, links, imagens e muito mais.
